
## SYNOPSIS
    ./pgedge ace mtree init CLUSTER_NAME <flags>

## DESCRIPTION
    Initialises the database with necessary objects for Merkle trees.

## POSITIONAL ARGUMENTS
    CLUSTER_NAME
        Name of the cluster.

## FLAGS
    -d, --dbname=DBNAME
        Name of the database.
    
    -n, --nodes=NODES
        Comma-separated subset of nodes.
    
    -q, --quiet_mode=QUIET_MODE
        Suppress output.
