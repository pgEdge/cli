
SYNOPSIS
    ./pgedge db COMMAND

COMMANDS
    COMMAND is one of the following:
     create              # Create a database owned by a specific user
     guc-set             # Set GUC
     guc-show            # Show GUC
     set-readonly        # Turn PG read-only mode 'on' or 'off'.
     test-io             # Use the 'fio' Flexible IO Tester on pg data directory
