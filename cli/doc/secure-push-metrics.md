## SYNOPSIS
    ./nodectl secure push-metrics CLUSTER_NAME TARGET_INFO <flags>
 
## DESCRIPTION
    push pgEdge Metrics to a specified target
 
## POSITIONAL ARGUMENTS
    CLUSTER_NAME
    TARGET_INFO
 
## FLAGS
    --client_id=CLIENT_ID
        Type: Optional[]
        Default: None
    --client_secret=CLIENT_SECRET
        Type: Optional[]
        Default: None
